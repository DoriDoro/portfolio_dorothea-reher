Python - Flask - GitHub - Vercel

This project is a 'work in project' and not yet done. Presentation of a selection of my projects with an "about me" page and my contact details.

Learning experience?

First project with Flask Framework for me. Creating two different routes, one for the index.html page and a dynamic route for all other .html pages inside the templates folder. (For the contact form I have created two different functions to store the message of the contact form inside a text file or a csv file. The final submit form route for the message is pointing to the csv file and all messages are stored there. The contact form is removed from this project now because it will be confusing.)

HTML, CSS and Bootstrap. Integrate images and resize them and organize and structure the given HTML. Understanding and using an HTML/CSS template with Bootstrap components. Usage of code of someone else to make it my own. I had a lot of fun creating that project.

The deployment at Vercel was a real challenge! The deployment on Heroku was tricky too, but on Vercel that was a real challenge. It took me hours and several attempts to deploy it on Vercel. I will try to deploy the other projects, if a deployment is possible, as fast as possible. The next challenge will be to deploy the Django projects with Vercel, yeah...
