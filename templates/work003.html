003 : Simple Blog

Python - Django - GitHub - Heroku

Single Page application for a simple blog. After sign in with the Testuser login, provided in the GitHub reposetory, you can add a blog article to the page.

Learning experience?

Login functionallity and display the username after successful login on website, in the navbar. Possibility of adding a blog just possible after successful login. List all blogs on the home page, using a for loop. 'Login' and 'Add Blog' is a Bootstrap modal fade class which is displaying a window to enter the necessary information.
