005 : DepEmp-Organizer

Python - Django - React - React-Bootstrap - GitHub - Heroku

This project is a Django React application. On the department page there is the department id, the name of the department and edit and delete option. With an add button you can add a new department to the list. Whereas on the employee page there are the names, departments, joining dates of the employee and the same options: edit, delete and add.

The app is not done yet and not yet deployed on Heroku or Vercel.

Learning experience?

To refresh my React knowledge and to combine the Django framework with it. This combination is a really great fit.
