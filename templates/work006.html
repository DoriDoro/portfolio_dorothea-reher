006 : LITRevu-project

Python - Django - CRUD - function-based-view - Boostrap5 - crispy-forms - GitHub

This project is a literature review website. User can register and post tickets for a book. The ticket has a title, a body and an image of a book. If a user created a ticket another user can write a review for this book. The user can follow other users, they can unfollow them and see which users are following the user. The tickets and reviews can be deleted or modified.

Learning experience?

Refresh my memory of create a Django app. I have now a better understanding ot the view-form concept. The dynamic between the view and the form is now clear.
